1. 설치 방법

이 앱은 웹 기반(PWA)이라 앱스토어 없이 바로 설치됩니다.

아이폰 (Safari)

  1. Safari에서 아래 주소 접속
    https://bboinzg-dev.github.io/daily-todo/
  2. 하단 공유 버튼 (□↑ 모양) 터치
  3. 메뉴를 아래로 스크롤 → "홈 화면에 추가" 터치
  4. 이름 확인 후 "추가" 터치
  5. 홈 화면에 앱 아이콘 생성!

안드로이드 (Chrome)

  1. Chrome에서 아래 주소 접속
    https://bboinzg-dev.github.io/daily-todo/
  2. 우측 상단 ⋮ 메뉴 터치
  3. "홈 화면에 추가" 또는 "앱 설치" 터치
  4. "설치" 확인
  5. 홈 화면에 앱 아이콘 생성!
설치 후에는 주소창 없이 전체화면 앱처럼 실행됩니다.

2. 시작하기 (PIN 설정)

첫 실행 시 4자리 PIN 번호를 설정합니다.

  1. 원하는 4자리 숫자 입력
  2. 확인을 위해 한 번 더 입력
  3. 설정 완료! 이후 앱을 열 때마다 PIN을 입력해야 합니다.
PIN을 잊어버리면 "초기화" 버튼으로 리셋할 수 있지만, 모든 데이터가 삭제됩니다. 반드시 백업해두세요.

3. 화면 구성

하단에 4개의 탭이 있습니다.

탭아이콘기능
오늘⏰대시보드, 습관, 루틴, 오늘 할일, 뽀모도로, 물마시기
할일✅전체 할일 목록, 주간 보기, 달력, 빠른 메모
기록✓해낸 일, 통계, 컨디션 기록, 엑셀 다운로드
설정⚙️임시보관, 알림, 눈 쉬기, 자동 잠금, 백업/복원

우측 하단의 + 버튼(FAB)을 누르면 새 할일을 추가할 수 있습니다.

4. 오늘 탭

앱을 열면 가장 먼저 보이는 메인 화면입니다. 오늘 해야 할 모든 것을 한눈에 확인합니다.

4-1. 컨디션 체크

화면 상단에 오늘의 기분을 이모지로 기록합니다.

이모지의미
😊좋음
🙂보통
😐그저그럼
😔별로
😫힘듦

기록 탭에서 최근 7일간의 컨디션 추이를 확인할 수 있습니다.

4-2. 대시보드

4개의 카드가 오늘의 현황을 요약합니다.

카드내용
할 일오늘 남은 할일 수
완료오늘 완료한 할일 수
습관오늘 습관 달성률 (%)
D-day가장 가까운 D-day까지 남은 일수

4-3. 뽀모도로 타이머

집중력을 높이기 위한 시간 관리 도구입니다.

사용법

  1. "시작" 버튼 → 25분 집중 타이머 시작
  2. 25분 완료 시 알림 → 자동으로 5분 휴식 시작
  3. 5분 휴식 완료 시 알림 → 1라운드 완료
  4. 반복! 오늘 몇 회 집중했는지 카운트됩니다.
"일시정지" 버튼으로 중단할 수 있고, "리셋"으로 초기화됩니다.

4-4. 물 마시기

하루 목표 8잔의 물 섭취를 추적합니다.

  • 💧 아이콘을 터치하면 마신 잔 수가 카운트됩니다.
  • 이미 채워진 잔을 터치하면 취소됩니다.
  • 알림 주기를 설정하면 정해진 시간마다 물 마시기 알림이 옵니다.
알림 주기설명
끄기알림 없음
30분30분마다 알림
1시간1시간마다 알림 (기본값)
1.5시간 / 2시간여유롭게 알림

4-5. 습관 트래커

매일 반복할 습관을 등록하고 체크합니다. 매일 자동 리셋됩니다.

  • 터치하면 완료 체크 / 다시 터치하면 취소
  • 연속으로 달성하면 🔥 연속 달성 배지가 표시됩니다.
  • "관리" 버튼으로 습관 추가/삭제 가능

습관 추가 시 설정 항목

  • 이름: 습관 이름 (예: 스트레칭)
  • 이모지: 아이콘 (예: 🧘)
  • 알림: 주기적 알림 (30분/1시간/2시간마다)

4-6. 고정 루틴

매일 고정으로 하는 일과를 관리합니다. 습관과 비슷하지만, 보다 구체적인 일과 항목입니다.

  • 매일 자동으로 리셋됩니다.
  • "관리" 버튼으로 추가/수정/삭제 가능
기본 루틴: 기상 후 물 한 잔, 스트레칭 10분, 오늘 할일 정리, 운동 30분, 독서 20분, 하루 회고 작성

4-7. 할 일 목록

오늘 해야 할 일과 며칠 내 해야 할 일이 표시됩니다.

  • 왼쪽 체크박스 터치 → 바로 완료 처리
  • 항목 자체를 터치 → 상세 정보 확인
  • 상세 화면에서 완료 / 보관 / 수정 가능

4-8. D-day

중요한 일정의 카운트다운을 표시합니다.

  • "추가" 버튼으로 D-day 등록
  • D-day 당일: D-DAY, 지난 후: D+N, 남은 날: D-N
  • 대시보드에 가장 가까운 D-day가 표시됩니다.

5. 할일 탭

모든 할일을 한곳에서 관리하고, 주간/월간 일정을 확인합니다.

5-1. 주간 보기

이번 주(월~일) 요일별 할일 현황을 한눈에 볼 수 있습니다.

  • 각 요일에 할일 수와 완료 수가 표시됩니다.
  • 카테고리별 색상 점으로 어떤 종류의 일이 있는지 파악 가능
  • 오늘 날짜는 테두리로 강조됩니다.

5-2. 달력

월간 달력에서 할일이 있는 날에 점(●)이 표시됩니다.

  • ◂ ▸ 버튼으로 월 이동
  • 날짜를 터치하면 해당 날짜에 바로 할일을 추가할 수 있습니다.

5-3. 빠른 메모

할일과 관련 없지만 급하게 적어야 할 내용을 기록합니다.

  • 내용 입력 후 "저장" 버튼 터치
  • 저장된 메모는 아래에 목록으로 표시
  • ✕ 버튼으로 삭제

6. 기록 탭

완료한 일의 기록과 통계를 확인합니다.

엑셀 다운로드

상단의 "해낸 일 엑셀 다운로드" 버튼을 누르면 완료일, 카테고리, 할일 이름, 상세 정보, 등록일이 포함된 CSV 파일이 다운로드됩니다. 엑셀에서 열 수 있습니다.

이번 주 완료율

요일별로 할일 완료율을 막대 그래프로 보여줍니다.

습관 연속 달성

각 습관의 연속 달성 일수를 🔥와 함께 표시합니다.

컨디션 기록

최근 7일간 기록한 컨디션 이모지와 라벨을 보여줍니다.

7. 설정 탭

임시보관

지금 당장 할 필요가 없어진 할일이 보관됩니다.

  • "복원" 버튼으로 다시 할일 목록으로 되돌릴 수 있습니다.
  • 완전히 불필요하면 상세 화면에서 삭제 가능

설정 항목

항목기능조작
🔔 알림 권한 허용푸시 알림을 받기 위한 권한 요청터치하여 허용
👀 눈 쉬기 알림설정된 주기마다 20초간 눈 휴식 화면 표시 (20-20-20 규칙)터치할 때마다 주기 변경 (10/15/20/30/45/60분/끄기)
🔒 자동 잠금미사용 시 자동으로 PIN 잠금 화면으로 전환터치할 때마다 시간 변경 (1/3/5/10/30분/끄기)
💾 데이터 백업모든 데이터를 JSON 파일로 내보내기터치하면 파일 다운로드
📂 데이터 복원백업 파일에서 데이터 가져오기터치하여 파일 선택

8. 할일 등록 / 관리

+ 버튼을 누르거나, 달력에서 날짜를 터치하면 새 할일을 등록합니다.

항목설명필수
할 일할일 내용✅
날짜예정일 (비워두면 기한 미정)
시간예정 시각
카테고리업무(파랑) / 개인(보라) / 건강(초록) / 공부(주황) / 기타(회색)
며칠 내 해야 함체크하면 '며칠 내' 섹션에 표시
반복매일 / 평일만 / 매주 / 매월 자동 반복
알림정시에 / 5분~1시간 전 알림
상세 정보준비사항, 관련 정보 등

할일 상태 흐름

할 일 → 완료 처리 → 해낸 일 (기록 탭)

할 일 → 보관 처리 → 임시보관 (설정 탭)

임시보관 / 해낸 일 → 복원 → 다시 할 일

오늘 날짜의 미완료 할일은 다음 날 자동으로 이월됩니다.
반복 할일은 완료 후 다음 해당일에 자동으로 새로 생성됩니다.

9. 알림 설정

알림을 받으려면 먼저 설정 탭 → "알림 권한 허용"을 터치해야 합니다.

알림 종류

알림조건
⏰ 할일 알림시간과 알림을 설정한 할일에 대해 지정 시각에 알림
💧 물 마시기설정된 주기마다 알림 (목표 미달성 시)
습관 알림습관에 알림 주기를 설정한 경우 주기마다 알림
🍅 뽀모도로집중 종료 / 휴식 종료 시 알림
👀 눈 쉬기설정된 주기마다 20초간 휴식 화면 표시
아이폰 Safari에서는 PWA 푸시 알림이 iOS 16.4 이상에서만 지원됩니다. 안드로이드 Chrome에서는 바로 작동합니다.

10. 데이터 백업 및 복원

백업 (내보내기)

  1. 설정 탭 → "💾 데이터 백업" 터치
  2. JSON 파일이 자동 다운로드됨
  3. 파일 이름: daily-todo-backup-날짜.json

복원 (가져오기)

  1. 설정 탭 → "📂 데이터 복원" 터치
  2. 이전에 백업한 JSON 파일 선택
  3. 확인 팝업에서 "확인" 터치
  4. 데이터 복원 완료 (앱 자동 새로고침)
복원 시 현재 데이터가 백업 파일의 데이터로 완전히 덮어씌워집니다.
주 1회 정도 백업해두면 폰을 바꾸거나 데이터 손실 시 복구할 수 있습니다.

11. 자주 묻는 질문 (FAQ)

Q. 앱이 안 열려요

인터넷 연결을 확인하세요. 최초 1회는 인터넷이 필요합니다. 이후에는 오프라인에서도 동작합니다.

Q. 업데이트는 어떻게 하나요?

자동입니다! 앱을 열면 최신 버전이 자동으로 적용됩니다. 데이터는 유지됩니다.

Q. 다른 기기에서 쓸 수 있나요?

백업 파일을 다른 기기로 옮기고 복원하면 됩니다. 자동 동기화는 지원하지 않습니다.

Q. PIN을 잊어버렸어요

잠금 화면 하단의 "초기화" 버튼을 누르세요. 모든 데이터가 삭제되고 처음부터 시작합니다. 평소에 백업해두면 복원 가능합니다.

Q. 알림이 안 와요

  1. 설정 탭 → "알림 권한 허용"을 눌렀는지 확인
  2. 폰 설정 → 해당 브라우저의 알림이 허용되어 있는지 확인
  3. 아이폰은 iOS 16.4 이상 + 홈 화면에 추가된 상태여야 합니다.

Q. 데이터가 사라졌어요

브라우저의 "사이트 데이터 삭제" 또는 "방문 기록 삭제"를 하면 데이터가 삭제될 수 있습니다. 정기적으로 백업하는 것을 권장합니다.

Q. 옛날 버전이 계속 보여요

브라우저 캐시를 삭제하고 다시 접속하세요. Chrome의 경우 설정 → 개인정보 → "캐시된 이미지 및 파일"만 삭제하면 데이터는 유지됩니다.
